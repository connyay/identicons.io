# identicons.io

Generate unique identicons from any string. Built with Rust and deployed on Cloudflare Workers.

## Usage

```html
<img src="https://identicons.io/your-username" alt="identicon">
```

Any path becomes a unique identicon - use usernames, emails, IDs, or any string.

## Development

```bash
# Run locally
npx wrangler dev

# Deploy
npx wrangler deploy
```
